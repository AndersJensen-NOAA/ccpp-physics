GFS_v15p2 Suite

Overview

Suite GFS_v15p2 has the parameterizations used in the GFS v15 implemented operationally in June 2019.

The GFS_v15p2 physics suite uses the parameterizations in the following order:

  • GFS_RRTMG
  • GFS_SFCLYR
  • GFS_NSST
  • GFS_OCEAN
  • GFS_NOAH
  • GFS_SFCSICE
  • GFS_HEDMF
  • GFS_UGWP_v0
  • GFS_RAYLEIGH
  • GFS_OZPHYS
  • GFS_H2OPHYS
  • GFS_SAMFdeep
  • GFS_SAMFshal
  • GFDL_cloud
